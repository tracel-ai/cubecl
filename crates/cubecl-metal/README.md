# CubeCL Metal Runtime

[CubeCL](https://github.com/tracel-ai/cubecl) Metal runtime.

## Platform support

| Platform   | GPU |
| :--------- | :-: |
| macOS 14+  | Yes |
| iOS 17+    | Yes |

## Limitations

- Metal 3+ required (atomic float and simdgroup matrix support)
- f64 not supported by Metal
